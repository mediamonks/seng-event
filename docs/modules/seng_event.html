    Calls all listeners on the given event listener map that should be called when the given event is dispatched. If no matching listeners are present, this function has no effect

    Type parameters

    • TEvent: AbstractEvent

    Parameters

    • listeners: EventListenerMap<TEvent>

      The object that contains listeners to call. Has the same format as the listeners property on EventDispatcher

    • event: TEvent

      The event that may trigger listeners in the map

    Returns boolean

    True if any of the listeners call [[IEvent.stopPropagation|stopPropagation()]] or [[IEvent.stopImmediatePropagation|stopImmediatePropagation]]. False if no listeners are called or none of them call [[IEvent.stopPropagation|stopPropagation()]] or [[IEvent.stopImmediatePropagation|stopImmediatePropagation]]

Const removeListenersFrom

  • removeListenersFrom<TEvent>(listeners: EventListenerMap<TEvent>, eventType?: TypesForEvent<TEvent> | null, handler?: EventHandlerForEvent<TEvent> | null, useCapture?: undefined | false | true): void
    • Defined in lib/EventDispatcher.ts:278

    Helper function for EventDispatcher.removeEventListener and EventDispatcher.removeAllEventListeners. Will remove all event listeners that match the given parameters from the given event listener map object. This function differs from removeEventListener() in that it does not use default values when you emit one of the parameters. Instead, it will remove event listeners of all possible values for that parameter.

    Type parameters

    • TEvent: AbstractEvent

    Parameters

    • listeners: EventListenerMap<TEvent>

      A map of listeners to remove from. See EventDispatcher.listeners

    • Optional eventType: TypesForEvent<TEvent> | null

      If set, will only remove listeners added with this eventType

    • Optional handler: EventHandlerForEvent<TEvent> | null

      If set, will only remove listeners with this handler

    • Optional useCapture: undefined | false | true

      If set, will only remove listeners with the same value for useCapture

    Returns void
